# constructor-shape-delete-highlight-fix-2026-04-02

Бэкап конструктора перед исправлением зависшей подсветки миниатюры фигуры после удаления shape-слоя.

## Содержимое

- `src-components-constructor/` — текущие компоненты конструктора на момент начала правки
- `src-hooks/useConstructorState.js` — актуальный state-хук конструктора

## Причина

После удаления активной фигуры в левой панели могла оставаться синяя подсветка миниатюры. Этот бэкап сохранён до внесения изменений в sidebar и страницу конструктора.
