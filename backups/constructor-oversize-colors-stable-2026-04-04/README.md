# constructor-oversize-colors-stable-2026-04-04

Rollback-срез актуальной стабильной версии конструктора после подключения реальных oversize mockup для нескольких цветов на 2026-04-04.

## Что сохранено

- src-components-constructor/ — все текущие компоненты конструктора
- src-hooks/useConstructorState.js — основная бизнес-логика состояния конструктора
- src-utils/constructor/ — утилиты конструктора, включая resize и shape helpers

## Для чего

Этот backup можно использовать как точку отката после подключения реальных PNG-мокапов oversize для чёрного, белого, розового, бежевого и серого цветов при сохранении общей size-aware геометрии и физических размеров print area.
